Problemă: Se consideră paralelipipedul dreptunghic cu dimensiunile L, l, h.
Exprimați înălțimea h cu ajutorul lui L și l, știind că:
$$\frac{1}{L + l} + \frac{1}{l + h} + \frac{1}{L + h} = \frac{1}{2} \cdot \left( \frac{1}{l} + \frac{1}{L} + \frac{1}{h} \right)$$
Sursa: Mate 2000 – Editura Paralela 45
Rezolvare:
Înmulțim ambele părți ale ecuației cu 4:
$$\frac{1}{L + l} + \frac{1}{l + h} + \frac{1}{L + h} = \frac{1}{2} \cdot \left( \frac{1}{l} + \frac{1}{L} + \frac{1}{h} \right) \Big| \cdot 4$$
$$\frac{4}{L + l} + \frac{4}{l + h} + \frac{4}{L + h} = \frac{2}{l} + \frac{2}{L} + \frac{2}{h}$$
Rearanjăm termenii:
$$\frac{1}{L} + \frac{1}{L} + \frac{1}{l} + \frac{1}{l} + \frac{1}{h} + \frac{1}{h} - \frac{4}{L + l} - \frac{4}{l + h} - \frac{4}{L + h} = 0$$
Grupăm termenii:
$$\left( \frac{1}{L} + \frac{1}{l} - \frac{4}{L + l} \right) + \left( \frac{1}{L} + \frac{1}{h} - \frac{4}{L + h} \right) + \left( \frac{1}{l} + \frac{1}{h} - \frac{4}{l + h} \right) = 0$$
Efectuăm calculele în prima paranteză:
$$\frac{1}{L} + \frac{1}{l} - \frac{4}{L + l} = \frac{l(L+l) + L(L+l) - 4Ll}{Ll(L+l)}$$
$$= \frac{Ll + l^2 + L^2 + Ll - 4Ll}{Ll(L+l)} = \frac{L^2 - 2Ll + l^2}{Ll(L+l)}$$
$$= \frac{(L-l)^2}{Ll(L+l)}$$
Procedând în același mod pentru celelalte paranteze, obținem:
$$\frac{(L-l)^2}{Ll(L+l)} + \frac{(L-h)^2}{Lh(L+h)} + \frac{(l-h)^2}{lh(l+h)} = 0$$
Deoarece avem numai numere nenegative, egalitatea are loc dacă și numai dacă fiecare termen este zero:
$$(L-l)^2 = 0 \text{ și } (L-h)^2 = 0 \text{ și } (l-h)^2 = 0$$
Deci:
$$L - l = 0 \text{ și } L - h = 0 \text{ și } l - h = 0$$
Concluzie: $$L = l = h$$ Paralelipipedul dreptunghic este de fapt un cub!
