Enunț:
Fie ABCDA'B'C'D' un paralelipiped dreptunghic cu AB = √2 cm și BC = √3 cm. Știind că o furnică parcurge distanța de la punctul B la punctul D' pe un drum de lungime minimă de √(11 + 2√6) cm, aflați înălțimea paralelipipedului.
Înțelegerea problemului
Avem un paralelipiped dreptunghic ABCDA'B'C'D' cu:

• AB = √2 cm
• BC = √3 cm
• Înălțimea h = AA' (necunoscută)

Furnica merge de la B la D' pe suprafața paralelipipedului, pe drumul cel mai scurt, care are lungimea √(11 + 2√6) cm.
Metoda de rezolvare
Pentru a găsi drumul minim pe suprafața paralelipipedului, trebuie să "desfășurăm" suprafața. Există mai multe posibilități de desfășurare, dar vom analiza cele mai probabile trasee.

Cheia este că drumul cel mai scurt pe o suprafață desfășurată este o linie dreaptă între cele două puncte.
Traseul 1: Prin fața laterală BCC'B' și apoi fața superioară
Desfășurând fețele ABB'A' și ADD'A', obținem un dreptunghi în care:

• Pornim din B
• Drumul minim este o linie dreaptă până la D'

În acest caz, coordonatele sunt:
• B la poziția (0, 0)
• D' la poziția (BC + CD, h) = (√3 + √2, h)

Distanța: d₁ = √[(√3 + √2)² + h²]
Calculul lui (√3 + √2)²
Să calculăm (√3 + √2)²:

(√3 + √2)² = (√3)² + 2·√3·√2 + (√2)²
= 3 + 2√6 + 2
= 5 + 2√6

Deci distanța devine:
d₁ = √(5 + 2√6 + h²)
Ecuația drumului minim
Știm că drumul minim este √(11 + 2√6) cm.

Prin urmare:
√(5 + 2√6 + h²) = √(11 + 2√6)
Rezolvarea ecuației
Ridicând ambele părți la pătrat:

5 + 2√6 + h² = 11 + 2√6

Simplificăm (scădem 2√6 din ambele părți):

5 + h² = 11

h² = 11 - 5

h² = 6

h = √6 cm
Verificare
Să verificăm rezultatul:

Distanța minimă = √(5 + 2√6 + 6) = √(11 + 2√6) ✓

Rezultatul este corect!
Răspuns final
Înălțimea paralelipipedului este h = √6 cm.
